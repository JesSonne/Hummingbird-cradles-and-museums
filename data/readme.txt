Description of data files:

biogeographical null model.R: R code used to conduct the biogeographical null model. See also methods for detailed description. 

Data.csv: data frame containing the raw and standardized richness of young and old endemic species for each grid cell. 

habitat_preferences.csv: habitat preferences for young and old endemic species.

Hummingbird age classes.csv: Data frame listing the hummingbird species identified as old and young endemic species. See 
supplementary material for additional information.

species elevations.csv: Data frame listing the minimum and maximum elevational range limits for each hummingbird species
